Описание системы DNS

1. Общие требования к интерфейсу:
- Поддержка русского языка
- Типизированные интерфейсы подсистем
- Шрифт: Montserrat (для основного текста и заголовков)
  - Основной текст: размер 16, цвет #333
  - Заголовки: размер 28

2. Цветовая палитра:
- Основной цвет фона: #F9F9F9
- Дополнительный цвет: #76926D
- Побочный цвет: #F4C991

3. Функциональные требования:
Система состоит из одного модуля для пользователя, который обеспечивает:
- Регистрацию в системе
- Обязательную авторизацию для доступа
- Управление данными о курсах (добавление и изменение)
- Поиск курсов по параметрам
- Запись на курсы
- Просмотр курсов
- Добавление курсов в избранное
- Просмотр личных данных

4. Пользовательские роли:
Пользователь платформы имеет доступ к:
- Просмотру личной информации и информации о курсах
- Манипуляциям с данными о курсах
- Поиску курсов по ключевым данным
- Записи на курсы

5. Требования к пользовательскому интерфейсу:
- Наглядное и интуитивно понятное представление информации
- Логичная навигация между разделами через меню
- Понятные навигационные элементы
- Интерфейс должен быть удобен для пользователей с базовыми компьютерными навыками
- Логотип заказчика в шапке основного экрана

6. Управление данными:
Система должна обеспечивать возможность создания, актуализации и контроля следующих сущностей:
- Пользователи
- Курсы
- Записи на курсы
