1 АНАЛИЗ ТРЕБОВАНИЙ К РАЗРАБАТЫВАЕМОЙ СИСТЕМЕ

При изучении требований было составлено техническое задание для разработки программного обеспечения магазина электроники DNS, в котором были выделены основные требования к системе.

Интерфейс системы должен поддерживать русский язык. Интерфейсы подсистем должны быть типизированы. Должен использоваться шрифт Montserrat (для основного текста и заголовков), цвет - #333, размер 16 - основной текст, выводимый системой на форму и размер 28 - заголовки, выводимые системой на форму.

Цветовая палитра должна состоять из трёх цветов:
1. Цвет фона - #F9F9F9.
2. Дополнительный цвет - #76926D.
3. Побочный цвет - #F4C991.

В шапке основного экрана должен использоваться логотип магазина DNS.

Интерфейс системы должен обеспечивать наглядное, интуитивно понятное представление структуры каталога товаров, быстрый и логичный переход к соответствующим разделам системы через меню.

Навигационные элементы интерфейса системы должны обеспечивать понимание пользователем их смысла и обеспечивать навигацию по разделам системы и отображать соответствующую информацию о товарах и их характеристиках.

Интерфейс системы должен быть рассчитан на пользователей, малознакомых со специальными техническими знаниями в области компьютерной техники.

Также должна быть обеспечена возможность создания, актуализации и контроля следующих сущностей:
- Товары
- Категории товаров
- Заказы

Требования к составу и структуре

Система должна состоять из одного модуля: для управления магазином электроники.

Функциональные требования

Модуль для управления магазином должен обеспечивать:
- Просмотр каталога товаров
- Поиск товаров по различным параметрам
- Фильтрацию товаров по категориям
- Просмотр детальной информации о товаре
- Добавление товаров в корзину
- Оформление заказа
- Просмотр истории заказов
- Управление личным кабинетом

Основные прецеденты и их описание

Пользователь системы - покупатель, имеющий доступ к:
- Просмотру каталога товаров и их характеристик
- Поиску товаров по различным параметрам
- Добавлению товаров в корзину
- Оформлению заказов
